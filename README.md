# qingflow-customer-mgmt

基于[轻流](https://qingflow.com)低代码平台搭建的客户进件、管理与审批系统设计方案。

## 项目概述

本方案覆盖客户生命周期管理的完整链路：

- **客户进件**：在线填单、资料上传、企业/个人双模式
- **审批流程**：四级审批链（客户经理 → 部门主管 → 总监 → 总经理），根据客户等级动态路由
- **客户管理**：客户档案、信息变更、跟进记录、分级管理
- **数据分析**：仪表盘、来源分析、审批时效监控

## 审批流程概览

```
客户经理审核 → 部门主管审批 → [总监审批] → [总经理审批]
                                  ↑              ↑
                            银卡及以上      金卡/钻石客户
```

## 角色权限

| 角色 | 核心权限 |
|---|---|
| 系统管理员 | 全量数据 + 系统配置 |
| 总经理 | 全量查看 + 最高审批 |
| 总监 | 本部门 + 审批（银卡+） |
| 部门主管 | 本部门 + 审批（普通） |
| 客户经理 | 自己的客户 + 发起申请 |
| 普通员工 | 受限查看 + 发起进件 |

## 技术栈

- **平台**：轻流 QingFlow（低代码）
- **数据存储**：轻流内置表单引擎
- **流程引擎**：轻流流程设计器
- **报表**：轻流报表中心
